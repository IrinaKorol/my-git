git clone путь-к-репозиторию- клонировать репозиторий
git status - статус текущего локального репозитория
git add путь к файлу - добавить новый файл под контроль ГИТа
git commit -am "added new file index.html"
git push - протолкнуть файл измененный на репозиторий
git pull - получить файл измененный с репозиторий
